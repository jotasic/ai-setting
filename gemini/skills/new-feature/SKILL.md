# Implement New Feature

Implements a new feature following a structured workflow.

## Arguments

- `$ARGUMENTS`: Feature description

## Workflow

### 1. Requirements Analysis
- Understand requirements
- Analyze codebase / impact

### 2. Design
- Implementation plan
- File structure
- API/Interface design

### 3. Implementation
1. Create feature branch
2. Implement core logic
3. Add error handling
4. Write tests
5. Update docs

### 4. Verification
- Run tests (Unit/Integration)
- Manual check

### 5. Cleanup
- Prepare for review
- Lint/Format

## Output
- Feature definition
- Files created/modified
- Tests added
- Next steps (Review/Merge)
