# MCP (Model Context Protocol) Demo

Guide to using MCP servers.

## Arguments

- `$ARGUMENTS`: Server name (optional)

## Available MCP Servers

### 1. Filesystem
Access local files.

### 2. GitHub
Interact with GitHub API.

### 3. PostgreSQL
Database queries.

### 4. Memory
Knowledge graph memory.

### 5. Puppeteer
Browser automation.

### 6. Sequential Thinking
Complex reasoning steps.

## Configuration

`gemini/mcp.json` or project specific `.gemini/mcp.json`.
