---
description: Bootstrap a new project environment
---

1. Create a standard directory structure:
    - `src/`: Source code
    - `tests/`: Tests
    - `docs/`: Documentation
    - `scripts/`: Utility scripts
2. Initialize git:
    - Run `git init`
    - Create a `.gitignore` file suitable for the project type (ask user for language if unknown)
3. Create a basic `README.md` with:
    - Project Title
    - Description
    - Installation instructions
4. Setup language-specific configuration:
    - Ask user for the primary language (e.g., Python, Node.js)
    - Create appropriate config files (e.g., `package.json`, `requirements.txt`)
