# Cursor AI Configuration

Cursor IDE를 위한 설정 및 규칙 예제입니다.

## Structure

```
cursor/
├── .cursorrules      # 프로젝트 규칙
├── prompts/          # 커스텀 프롬프트
└── README.md
```

## Setup

프로젝트 루트에 `.cursorrules` 파일을 생성하세요.

## Key Files

- `.cursorrules`: 프로젝트별 AI 동작 규칙
- `~/.cursor/settings.json`: 전역 설정
